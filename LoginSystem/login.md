# Login

## Para dar Login

- precisamos de 2 informações

>Nº do cartão da escola

>Password

## informação que precisamos para criar user

- **1. nº cartão**
- **2. nome**
- **3. apelido** 
- **4. idade** 
- **5. email** 
- **6. password** 
- **7. número** 
- **8. nivel**

## Base de dados

**1. Tabela: user**

- número do cartão (id) [INT 5]
- nome do aluno [TEXT 15]
- apelido do aluno [TEXT 25]
- idade [INT 2]
- email [TEXT 80]
- Password [TEXT 50]
- número Turma [INT 2]
- nivel [INT 1]

**2. Tabela: Calendario**
    
**3. Tabela: Turma**
